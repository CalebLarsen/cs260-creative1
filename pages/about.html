What is Unum?

Unum is a programming language that unites many systems into one.

Features of Unum

  • Unthreads which allow a Unum program to run equally seamlessly on 1 system as on 100

  • Blazing-fast garbage collection ran locally on each node of your unumverse

  • Simple human-readable syntax

  • Short compilation times to save programmer sanity

This allows the average programmer to quickly write programs that can run on their machine as easily as a massively distributed system.
